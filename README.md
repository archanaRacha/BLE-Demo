# BLE Demo (SwiftUI)
A simple Bluetooth Low Energy (BLE) scanner built with SwiftUI and CoreBluetooth.

## Features
- Scans for nearby BLE devices
- Displays name and signal strength (RSSI)
- Built using MVVM architecture

## Tech Stack
- SwiftUI
- CoreBluetooth
- Combine
